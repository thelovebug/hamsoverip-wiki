# HamVoIP NL Network

HamVoIP NL has a trunk line connected to the Hams Over IP network, which allows users from both networks to call each other.

Questions about the HamVoIP NL Network can be directed to the HamVoIP NL team directly, via their [Contact page](https://www.hamvoip.nl/contact.php).

For more information about HamVoIP NL Network, go to [HamVoIP NL Website](https://www.hamvoip.nl/).

## Calling a HamVoIP NL user from Hams Over IP

To call a HamVoIP NL number from the Hams Over IP network, dial `031`, and the 3- or 4-digit number of the user you wish to call. Your caller ID from the Hams Over IP side will show on the users phone, allowing them to call you back or hit redial.

* Try it: dial `0316001`. You should reach the ARRL Audio News.

## Calling a Hams Over IP Number from HamVoIP NL Network

To call a Hams Over IP number from the HamVoIP NL Network, dial `577` and the 6-digit Hams Over IP number of the user you wish to call. Your caller ID from the HamVoIP NL side will show on the users phone, allowing them to call you back if needed.

## HamVoIP NL Network White Pages

Don't know the number of the person you are trying to call on the HamVoIP NL Network? Look them up in the [HamVoIP NL Phonebook](https://www.hamvoip.nl/numbers_public.php).
